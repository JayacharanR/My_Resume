## 🚀 Overview

This repository contains the source code for my resume, maintained as **"Resume as Code"**. 
By separating content (YAML) from presentation (Typst/Markdown), I ensure my CV is always version-controlled, consistent, and easy to update.

## 📂 Repository Structure

- **`Jayacharan_R.yaml`**: The single source of truth for my resume content.
- **`rendercv_output/`**: Contains the generated artifacts (PDF, Markdown, HTML).
- **`classic/` & `markdown/`**: Custom Jinja2 templates used to style the output.

## 🛠️ How It Works

1.  **Content**: All data (Education, Experience, Skills) is defined in [`Jayacharan_R.yaml`](Jayacharan_R.yaml).
2.  **Engine**: [RenderCV](https://github.com/sinaatalay/rendercv) processes the YAML.
3.  **Output**: Generates PDF, HTML, and Markdown files in the `rendercv_output/` directory.

## 💻 Local Build Instructions

To generate the resume on your local machine:

```bash
# 1. Install RenderCV
pip install rendercv

# 2. Render the resume
rendercv render Jayacharan_R.yaml
```
